.. _custom_fluid_templates:

Use your own fluid templates
############################

To use custom fluid templates you can set following constants:

* plugin.tx_ceheaderimage.view.templateRootPath
* plugin.tx_ceheaderimage.view.partialRootPath
* plugin.tx_ceheaderimage.view.layoutRootPath
